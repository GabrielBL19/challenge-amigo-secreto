# challenge-amigo-secreto
<h1>Amigo secreto mediante sorteo</h1>
Este es un juego para sortear entre varios nombres de amigos ingresados en la caja de texto, se sortean al azar y uno es elegido.
Puedes utilizar este proyecto para fiestas, cumpleaños, festividades, rifas, etc. Es muy accesible y funcional. <br>

Esta es su página principal.

Luego click en el botón "añadir" y se añade el nombre a la lista.
<img width="1869" height="925" alt="image" src="https://github.com/user-attachments/assets/1115db6e-7dca-4217-8f05-9978ab8e56f7" />
